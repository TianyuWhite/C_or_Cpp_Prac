7-3 字符转换（15 分）
本题要求提取一个字符串中的所有数字字符（'0'……'9'），将其转换为一个整数输出。

输入格式：
输入在一行中给出一个不超过80个字符且以回车结束的字符串。

输出格式：
在一行中输出转换后的整数。题目保证输出不超过长整型范围。

输入样例：
free82jeep5
输出样例：
825


#include <stdio.h>
#include <stdlib.h>

int main()
{
    int i, j=0;
    char s[90];
    char s0[90];
    gets(s);
    for(i=0;i<strlen(s);i++){
        if(isdigit(s[i])){
            s0[j] = s[i];
            j++;
        }
    }
    printf("%ld", atol(s0));
    return 0;
}
