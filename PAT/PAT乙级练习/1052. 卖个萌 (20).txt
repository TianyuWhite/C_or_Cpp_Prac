萌萌哒表情符号通常由“手”、“眼”、“口”三个主要部分组成。简单起见，我们假设一个表情符号是按下列格式输出的：

[左手]([左眼][口][右眼])[右手]
现给出可选用的符号集合，请你按用户的要求输出表情。

输入格式：

输入首先在前三行顺序对应给出手、眼、口的可选符号集。每个符号括在一对方括号[]内。题目保证每个集合都至少有一个符号，并不超过10个符号；每个符号包含1到4个非空字符。

之后一行给出一个正整数K，为用户请求的个数。随后K行，每行给出一个用户的符号选择，顺序为左手、左眼、口、右眼、右手――这里只给出符号在相应集合中的序号（从1开始），数字间以空格分隔。

输出格式：

对每个用户请求，在一行中输出生成的表情。若用户选择的序号不存在，则输出“Are you kidding me? @\/@”。

输入样例：
[r][q][o][~\][/~]  [<][>]
 [s][t][^][-][=][>][<][@][⊙]
[Д][][_][ε][^]  ...
4
1 1 2 2 2
6 8 1 5 5
3 3 4 3 3
2 10 3 9 3
输出样例：
r(st)q
<(@Д=)/~
o(^ε^)o
Are you kidding me? @\/@


#include <iostream>
#include <bits/stdc++.h>

using namespace std;

char hand[20][30];
char eye[20][30];
char mouth[20][30];

int getemotion(char p[][30]){
    int i=0, j=0, flag=0, count=0;
    char c;
    while((c=getchar())!='\n'){
        if(c=='['){
            flag=1;
            i++;
            j=0;
            count++;
        }
        else if(c==']'){
            flag=0;
            p[i][j]='\0';
        }
        else{
            if(flag==1){
                p[i][j++] = c;
            }
        }
    }
    return count;
}

int main()
{
    int handl, eyel, mouthl, n;
    int h1, e1, m, e2, h2;
    handl = getemotion(hand);
    eyel = getemotion(eye);
    mouthl = getemotion(mouth);
    cin>>n;
    for(int i=0;i<n;i++){
        scanf("%d %d %d %d %d", &h1, &e1, &m, &e2, &h2);
        if( (h1>handl) || (h2>handl) || (e1>eyel) ||(e2>eyel) || (m>mouthl) || (h1<1) || (h2<1) || (m<1) || (e1<1) || (e2<1)){
            printf("Are you kidding me? @\\/@\n");
        }
        else{
            printf("%s(%s%s%s)%s\n", hand[h1], eye[e1], mouth[m], eye[e2], hand[h2]);
        }
    }
    return 0;
}
